艾森豪威尔矩阵是一款简洁、专注的四象限任务管理工具，帮助你根据“重要性”和“紧急性”重新整理每天的任务优先级，并把复杂任务拆解成更容易执行的行动步骤。

当任务越来越多时，真正困难的往往不是记录任务，而是判断：今天到底应该先做哪一件？哪些任务需要立即处理？哪些任务值得长期投入？哪些事务只是看起来紧急，却并不真正重要？

艾森豪威尔矩阵通过四个清晰的任务区域，帮助你把任务分为：

• 重要且紧急：需要立即处理的事项  
• 重要但不紧急：值得长期投入和计划推进的事项  
• 不重要但紧急：需要快速处理、委托或减少干扰的事项  
• 不重要也不紧急：可以延后、删除或限制时间的事项  

通过这种方式，你可以更快看清任务结构，减少被琐事牵着走，把精力放在真正重要的事情上。

更重要的是，艾森豪威尔矩阵不仅帮助你判断任务优先级，也帮助你更有效地完成任务。

本 App 引入了借鉴 project management 原则的子任务管理功能。对于复杂任务，你可以将一个大任务拆分为多个更小、更明确的子任务，并分别设置每个子任务的完成进度。主任务会根据子任务进度自动汇总整体完成情况，让你不再只看到“完成 / 未完成”这两个状态，而是可以清楚看到任务正在推进到哪一步。

这种任务拆分方式适合学习计划、工作项目、申请材料准备、健身目标、长期职业规划等需要持续推进的事项。通过把大任务拆成可执行的小步骤，你可以减少拖延，提高执行效率，也更容易保证最终完成质量。

主要功能：

【四象限任务管理】
以四象限视图展示不同优先级的任务，让你直观看到每类任务的数量、状态和当前压力。

【任务进度】
每个任务都可以设置完成进度。相比普通待办清单只区分“完成 / 未完成”，进度功能可以帮助你更真实地记录任务推进情况。

【子任务管理】
复杂任务可以拆分为多个子任务。每个子任务都可以单独设置完成进度，主任务会根据子任务自动汇总整体进度。这个功能借鉴了 project management 中的任务分解思路，帮助你把模糊的大目标转化为清晰、可执行、可追踪的小步骤。

【逾期任务统计】
首页会清晰显示逾期任务数量，帮助你及时发现已经超过截止时间但尚未完成的事项。

【已完成任务页面】
已完成任务会被集中整理到独立页面，方便你回顾已经完成的工作，也可以在需要时恢复为未完成状态。

【色盲友好模式】
内置色盲友好配色方案，并配合图标和文字标签共同区分任务类别，减少单纯依赖颜色带来的识别困难。

【本地数据保存】
当前版本不需要登录，不使用广告，不使用第三方分析。任务数据默认保存在本机，适合注重隐私和简洁体验的用户。

适合人群：

• 希望提升每日任务优先级判断的人  
• 经常被紧急事务打断的人  
• 想把长期重要目标持续推进的人  
• 需要把复杂任务拆解成多个步骤的人  
• 想追踪任务完成进度，而不只是记录待办事项的人  
• 喜欢简洁、本地、无账号任务管理工具的人  
• 希望用四象限方法整理工作、学习和生活任务的人  

艾森豪威尔矩阵不是为了让你记录更多任务，而是帮助你更清楚地决定：什么值得现在去做，什么应该计划推进，什么可以减少，什么可以放下。

同时，它也帮助你把真正重要的任务拆解成可执行的步骤，让任务不只是被记录下来，而是被持续推进、逐步完成。
